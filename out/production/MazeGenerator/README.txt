Description: This Project generates a maze using a 2d array after its been generated using bit arithmetic and writing to a file.
then parsing the output file a random path is chosen by removing walls , with a random left starting point and a random finishing point on
the right.

To run this file follow use the following arguments in the terminal:
MazeGen.java <width> <height> <output file> then hit enter

after running you can drag the output file into the maze generator on ezekiel to see the finished product.
